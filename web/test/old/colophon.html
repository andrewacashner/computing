Technology

The code is validated HTML5 and CSS3, and is continually tested on all available graphical and text browsers, including mobile versions. The site requires HTML5 video and audio to function correctly. Some HTML5 features are disabled in mobile browsers and text browsers, and some other browser do not support the standard fully.

The site was built with free and open-source software on the Debian GNU/Linux operating system. This page is hosted through GitHub.

The fonts used are Linux Libertine Display (serif) and Linux Biolinum (sans serif) from the Libertine Open Fonts Projekt. They are free and legal to distribute and modify under a SIL Open Font License.
