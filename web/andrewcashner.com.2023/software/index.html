Projects (Code on GitHub)

  • Chronoquiz: Build and share your own historical timeline quiz games (Typescript/React front end, Python/Django/MySQL backend)
  • Arca musarithmica: Generate your own seventeenth-century music based on your own chosen texts, styles, and moods with this implementation of a computing device for automatic music composition invented by Athanasius Kircher in 1650 (Javascript front end, PHP/Haskell back end)
  • Lirio: A library for the Lilypond music-notation software enabling beautiful and rigorous typesetting of early modern European music with a configurable semantic interface (Lilypond and Scheme)
  • ly2mei, Compiles Lilypond files written using the Lirio library to MEI-XML (in Object Pascal)
  • LaTeX packages: semantic-markup, musicography, octave
